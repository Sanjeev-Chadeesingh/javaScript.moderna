**array callback methods
    *forEach -runs a cb for each element in the array, doesn't create a new value / array

    *map -runs a cb on each element in the array, puts the newly made elements into a new array
    
    *find -runs a boolean cb, returns the first element that satisfies the boolean
    
    *filter -creates a new array subset, of elements satisying the boolean cb
    
    *every -tests whether all elements in an array satisfy the provided boolean cb
    
    *some -tests whether some elements in an array satisfy the provided boolean cb
    
    *sort -mutates original array via a comparing function, can use slice to make a new array
    
    *reduce -reduces an entire array down to a single value via a reducer function
