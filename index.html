Numbers In Javascript

Javascript has ONE number type

does not store precise numbers (think a buncha decimals)

clear() → clears the javascript console

basic math operations work in javascript

Modulus % operator → operand1 % operand2

takes the second operand and divides it into the first, returning the remainder that's left

25 % 5 → returns 0

14 % 4 → returns 2

Order Of Operations Applies To All Math Operations In Javascript

NaN → special value, stands for Not a Number, is considered a numeric value

VARIABLES

Variables are like "labeled jars" for values in javascript

let → is mutable, can change after declaration

let someName = value;

Numeric Variable Mutation:

let age = age + 3;

let age += 3;

let age++ → incrementing by one

let age-- → decrementing by one

Variable Naming Standard In Javascript → camel case → camelCase

const → is immutable, cannot change after declaration

const someName = value;

const works like let, except you cannot change the value

const test = 7;

test += 7; → will not work, const is immutable!!

var → Works like let. Still generally works, just old, and not ideal. Better to use let+const

Variables in javascript can change type

let someName = someString;

let someName = someNumber;

Booleans → true or false

Strings → textual values, strings of characters, wrap them in quotes

let firstName = 'Sanjeev';

let lastName = 'Chadeesingh';

let fullName = firstName + ' ' + lastName;

Whether you use single or double quotes, just make sure you are consistent and close them correctly

Strings are indexed, left to right, starting from zero. Spaces and special characters are included in the index

Every string has a property called length

let firstName = 'Sanjeev';

firstName.length = 7;

firstName[0] = 'S';

String Methods

Strings come with a set of built-in methods, which are actions that can be performed on or with that particular string

We can do things like → search within a string, replace parts of a string, change case (upper/lower)

string.method()

let firstName = 'Sanjeev';

firstName.toUpperCase();

firstName = 'SANJEEV';

The above line is not a new string. Strings are immutable. You'd have to save the newly changed string to a newly made variable
